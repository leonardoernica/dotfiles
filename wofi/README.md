# Wofi

Wofi está instalado e sendo usado como launcher de aplicações (definido em `hypr/hypr/hyprland.conf` como `$menu = wofi --show drun`), mas atualmente não há configurações customizadas.

## Configuração

Se você quiser criar configurações customizadas para o Wofi, você pode criar:

- `~/.config/wofi/config` - Arquivo de configuração principal
- `~/.config/wofi/style.css` - Estilos CSS customizados

O Wofi pode funcionar sem arquivos de configuração, usando os padrões, mas você pode customizar conforme necessário.

Para mais informações, consulte: `wofi --help`
